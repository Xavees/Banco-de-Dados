# 🗄️ Banco de Dados — Estudos e Práticas

Repositório dedicado aos estudos de Banco de Dados Relacionais, modelagem de dados e linguagem SQL, desenvolvido durante minha formação em Tecnologia da Informação.

O conteúdo deste repositório é baseado principalmente no curso ministrado pelo **Professor Jorge Sant Ana**, abordando desde os conceitos fundamentais de modelagem até recursos avançados presentes nos principais Sistemas Gerenciadores de Banco de Dados (SGBDs).

---

# 📚 Objetivo do Repositório

Este repositório tem como objetivo documentar minha evolução nos estudos de Banco de Dados, armazenando exercícios, exemplos, modelos, consultas SQL e projetos desenvolvidos ao longo do aprendizado.

Além disso, serve como material de consulta para reforçar conceitos importantes utilizados no desenvolvimento de sistemas e aplicações que dependem do armazenamento e gerenciamento de dados.

---

# 🧠 Conteúdos Estudados

Durante o curso foram abordados temas essenciais para o desenvolvimento de Bancos de Dados Relacionais, incluindo:

## Modelagem de Dados

* Modelagem Conceitual;
* Modelagem Lógica;
* Modelagem Física;
* Entidade e Relacionamento (MER);
* Diagrama Entidade-Relacionamento (DER);
* Cardinalidades;
* Chaves Primárias e Estrangeiras.

## Linguagem SQL

* DDL (Data Definition Language);
* DML (Data Manipulation Language);
* DQL (Data Query Language);
* T-SQL;
* Criação e gerenciamento de tabelas;
* Inserção, atualização e exclusão de dados;
* Consultas e filtros.

## Normalização

* Primeira Forma Normal (1FN);
* Segunda Forma Normal (2FN);
* Terceira Forma Normal (3FN);
* Eliminação de redundâncias;
* Integridade dos dados.

## Consultas Relacionais

* INNER JOIN;
* LEFT JOIN;
* RIGHT JOIN;
* FULL JOIN;
* Relacionamentos entre tabelas;
* Consultas avançadas.

## Recursos Avançados

* Views;
* Triggers;
* Procedures;
* Functions;
* Ambientes transacionais;
* Controle de integridade.

## Documentação

* Dicionário de Dados;
* Documentação de tabelas;
* Padronização de estruturas.
